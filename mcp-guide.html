Claude Code를 외부 도구에 연결하기

지금까지 Claude Code는 내 컴퓨터 안에서만 일했습니다.
이제 Slack, Gmail, Notion 같은 실제 쓰는 도구에 연결해서, 결과물이 자동으로 전달되게 만듭니다.
개념
MCP가 뭔가

MCP (Model Context Protocol)는 Claude Code를 외부 서비스에 연결하는 기술입니다.
어렵게 생각할 필요 없습니다. 연결 플러그라고 보면 됩니다.

MCP 없이
Claude가 리서치 완료 → 내가 결과 복사 → Slack에 붙여넣기
매번 수동
MCP 연결 후
Claude가 리서치 완료 → Slack에 자동 전송
완전 자동
💡
연결 가능한 서비스 예시: Slack, Gmail, Google Calendar, Notion, Linear, GitHub 등
Anthropic이 공식 커넥터를 계속 추가하고 있습니다. 이 가이드에서는 가장 실용적인 Slack 연동을 먼저 다룹니다.
사전 준비
시작 전 확인
필요한 것:
✅ Claude Code 설치 완료 (1단계)
✅ 서브 에이전트 설치 완료 (2단계)
✅ Slack 워크스페이스가 있어야 합니다 (무료 플랜도 가능)
✅ Slack에 결과를 받을 채널이 있어야 합니다
📌
2단계에서 만든 스케줄 에이전트와 연동하면 "매일 자동 리서치 → Slack 전송"까지 완전 자동화가 가능합니다.
실전
스케줄 리서치 → Slack 자동 전송

완성되면 이렇게 작동합니다:

매일 오전 9시
→
자동 리서치 실행
→
Slack 채널에 자동 전송
1
Slack MCP 커넥터 연결
~5분
▶
Claude Code가 Slack에 메시지를 보낼 수 있도록 연결합니다.
Step 1. claude.ai/settings/connectors 접속
Step 2. "Add Connector" 클릭 → Slack 선택
Step 3. Slack 워크스페이스 로그인 → 권한 허용
Step 4. 연결 완료되면 커넥터 목록에 Slack이 표시됨
⚠️
커넥터 연결은 claude.ai 웹에서만 가능합니다. VSCode 안에서는 할 수 없습니다.
연결 후에는 VSCode의 Claude Code에서 바로 사용 가능합니다.
2
스케줄 에이전트에 Slack 연결
~5분
▶
2단계에서 만든 스케줄 에이전트에 Slack 전송 기능을 추가합니다. 새로 만들어도 되고, 기존 스케줄을 수정해도 됩니다.
새로 만드는 경우
VSCode의 Claude Code에 입력
매일 오전 9시에 우리 업계 주요 뉴스와 경쟁사 동향을 리서치해서 정리해줘. 포함할 내용: - 주요 뉴스 3개 (출처 명시) - 경쟁사 동향 - 투자 소식 - 시사점 결과를 Slack #데일리-리서치 채널로 보내줘. 이걸 스케줄로 등록해줘.
기존 스케줄을 수정하는 경우
VSCode의 Claude Code에 입력
기존 스케줄 에이전트를 수정해줘. Slack 커넥터를 연결하고, 결과를 Slack #데일리-리서치 채널로 자동 전송하도록 설정해줘.
💡
채널명을 정확히 입력하세요. Claude가 자동으로 채널을 찾아서 연결합니다.
채널이 Private인 경우, Claude 앱을 해당 채널에 먼저 초대해야 할 수 있습니다.
3
테스트 실행
~3분
▶
스케줄 등록 후 즉시 실행해서 Slack에 결과가 잘 도착하는지 확인합니다.
VSCode의 Claude Code에 입력
방금 등록한 스케줄을 지금 바로 실행해줘. 테스트야.
✅ 성공이면: Slack 채널에 리서치 결과가 도착
❌ 실패하면: 아래를 확인하세요

  1. claude.ai/settings/connectors에서 Slack이 연결되어 있는지
  2. 채널명이 정확한지 (오타 확인)
  3. Private 채널이면 Claude 앱이 초대되어 있는지
4
관리 및 확인
상시
▶
등록된 스케줄은 웹에서 관리합니다.
할 일 방법
실행 결과 확인 claude.ai/code/scheduled 접속
스케줄 수정 Claude Code에서 "스케줄 수정해줘"
스케줄 삭제 관리 페이지에서 직접 삭제
즉시 실행 "지금 바로 실행해줘"
응용
이런 것도 가능합니다

Slack 연동은 시작일 뿐입니다. MCP 커넥터가 추가되면 더 많은 자동화가 가능해집니다.

매일 아침 시장 브리핑 — 리서치팀이 매일 업계 뉴스를 조사하고, Slack #데일리-리서치 채널로 전송
매주 경쟁사 모니터링 — 매주 월요일 경쟁사 동향을 정리하고 Slack으로 공유
매월 투자 동향 리포트 — 매월 1일 업계 투자 소식을 정리해서 Slack으로 전송
🔮
향후 추가 예정: Gmail 연동 (자동 이메일 발송), Google Calendar 연동 (일정 자동 등록), Notion 연동 (문서 자동 저장) 등
Anthropic이 새 커넥터를 추가하면 이 가이드도 업데이트됩니다.

한상
스타트업 대표이사
스타트업 대표로 살다 보면 해야 할 일은 많고, 혼자 감당해야 하는 것도 많습니다.
시장 리서치, 계약서 검토, 개발팀에 넘길 PRD 작성까지 — 대표가 직접 챙겨야 하는 것들을 Claude Code로 하나씩 해보고 있습니다.

이 가이드는 그 과정을 기록한 것입니다. 써보면서 새로 알게 된 것, 더 잘 되는 방법을 계속 업데이트할 생각입니다.
